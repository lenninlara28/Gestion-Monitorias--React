EN EL ARCHIVO COMPRIMIDO SE ENCUENTRAN LAS CARPETAS
- MONITORIAS, que es donde esta alamacenado todo el frontend
- SERVER, donde se encuentra todo el Back-End

PASOS PARA EJECUCION:

1- DESCOMPRIMIR ZIP
2- ABRIR NUESTRO MOTOR DE BASES DE DATOS PARA IMPORTAR LA DB CORRESPONDIENTE
   LLAMADA gestion_monitorias.sql
3- DENTRO DE MYSQL IMPORTAMOS LA BASE DE DATOS
4- POR MEDIO DE LA TERMINAR O LA VENTANA DE COMANDOS NOS VAMOS A NUESTRO SEVIDOR
   EN AL CARPETA SERVER PARA INICIARLO
   UNA VEZ INICIADO ESTE DEBE CORRER SOBRA EL LOCALHOST:3001
5- POR MEDIO DE LA TERMINAL DE COMANDOS NOS VAMOS A LA CARPETA monitorias Y CON EL COMANDO 
   npm start DEBEMOS COMPILAR TODO EL CODIGO

6- UNA VEZ FUNCIONANDO EJECUTADO POR EL PUERTO 3000
	NOS ENCONTRAMOS CON UNA VENTANA PRINCIPAL DONDE PODEMOS EMPEZAR A GESTINAR EL PROYECTO
	PODEMOS ACCEDER A GESTINAR MONITORES Y GESTINAR LAS MONITORIAS
